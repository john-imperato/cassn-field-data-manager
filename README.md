# CA-SSN Field Data Manager

A Python desktop application for streamlined wildlife monitoring data collection and management. 
Designed for the University of California Natural Reserve System (UCNRS) California Sentinel Sites for Nature (CASSN) team working with camera traps and acoustic recorders across California.

Features:
• Guided workflow for multi-plot, multi-device data collection
• Automatic file renaming with standardized naming conventions
• EXIF metadata extraction from images 
• SHA-256 file hashing for data integrity verification
• CSV metadata generation
• Automatic upload to Box cloud storage with progress tracking
• Support for images and audio files
• Local staging with organized folder structure
